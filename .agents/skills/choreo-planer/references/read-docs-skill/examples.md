## Examples (Read Docs)

Example 1: Read a local README and explain setup
- Input: doc_path: README.md from the repo
- Output: A concise overview of the project, its prerequisites, and step-by-step setup from the README.

Example 2: Read an online API docs page
- Input: source_url: https://example.com/api/docs
- Output: Key concepts, endpoints, authentication notes, and a short onboarding steps list.

Example 3: Read a Docker Compose guide and extract dev-environment steps
- Input: doc_content containing a Docker Compose snippet
- Output: Summary of services, how to start, and prerequisites.
